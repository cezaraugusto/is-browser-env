# is-browser-env

> Checks whether or not the current runtime environment is a browser.

## Useful when

You want to check if a given condition needs to have browser support.

## Installation

```sh
$ npm i is-browser-env
```

## Usage
s
```js
import isBrowserEnv from 'is-browser-env'

if (isBrowserEnv) {
  // ...
}
```
